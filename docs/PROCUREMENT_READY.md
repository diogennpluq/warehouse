## 🚀 Как протестировать

### 1. Запустите бэкенд:
```bash
cd backend
go run main.go
```

### 2. Запустите фронтенд:
```bash
cd frontend
npm start
```

### 3. Пройдите по шагам:
1. Откройте http://localhost
2. Войдите в систему
3. Нажмите **"44-ФЗ"** в меню (оранжевая ссылка)
4. Нажмите **"📋 Создать закупку"**
5. Заполните 4 шага Wizard:
   - Шаг 1: Название, обоснование, сроки
   - Шаг 2: Товары, количество, КТРУ/ОКПД2
   - Шаг 3: 3 коммерческих предложения (цены)
   - Шаг 4: Тип процедуры, обеспечение
6. Нажмите **"Сгенерировать документы"**
7. Скачайте ZIP-архив

### 4. Проверьте документы:
Распакуйте ZIP и откройте файлы:
- `.docx` файлы должны отображаться с правильным русским текстом
- `.xlsx` файл должен содержать таблицу с ценами

## 🔍 Проверка кодировки

Если в документах всё ещё "кракозябры":

1. **Проверьте заголовки HTTP:**
   ```
   Content-Type: application/zip
   Content-Disposition: attachment; filename*=UTF-8''Zakupka.zip
   ```

2. **Проверьте XML документов:**
   ```xml
   <?xml version="1.0" encoding="UTF-8" standalone="yes"?>
   ```

3. **Откройте файл в текстовом редакторе:**
   - Notepad++ → Кодировки → UTF-8
   - VS Code → Правый нижний угол → UTF-8
