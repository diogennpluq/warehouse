### Пользовательский сценарий

1. **Шаг 1: Инициация**
   - Ввод названия и обоснования закупки
   - Выбор ответственного и комиссии
   - Указание адреса и сроков поставки

2. **Шаг 2: Техническое задание**
   - Добавление объектов закупки (позиций)
   - Указание кодов КТРУ/ОКПД2
   - Добавление характеристик товаров
   - Установка гарантийного срока

3. **Шаг 3: Расчет НМЦК**
   - Ввод данных из 3-х коммерческих предложений
   - Автоматический расчет средней цены
   - Вычисление коэффициента вариации (V)
   - Валидация по 44-ФЗ (V ≤ 33%)
   - Подсветка некорректных цен

4. **Шаг 4: Настройки процедуры**
   - Выбор типа: Электронный аукцион / Запрос котировок
   - Преимущества для СМП/СОНКО
   - Обеспечение заявки (0.5-1% от НМЦК)
   - Обеспечение контракта (5-30% от НМЦК)
   - Размер аванса (0-100%)

5. **Результат**

## 🔧 Технические детали

### Математика НМЦК (44-ФЗ)

```typescript
// 1. Средняя цена
averagePrice = Σ(prices) / n

// 2. Среднее квадратичное отклонение
σ = √(Σ(price - avg)² / (n-1))

// 3. Коэффициент вариации
V = (σ / averagePrice) × 100

// 4. Проверка однородности цен
isValid = V ≤ 33%

// 5. Итоговая НМЦК
NMCC = averagePrice × quantity
```

## 🚀 Как запустить

### Полная сборка

```bash
# 1. Поднять все сервисы
docker compose up -d

# 2. Приложение доступно по адресам:
# - Фронтенд: http://localhost:80
# - Бэкенд API: http://localhost:8080
# - PostgreSQL: localhost:5432

# 3. Войти в систему
# Логин: admin
# Пароль: admin123
```

**Backend:**
```bash
cd backend
go mod download
go run main.go
```

**Frontend:**
```bash
cd frontend
```
