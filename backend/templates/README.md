# Шаблоны документов для закупок по 44-ФЗ

## Обзор

В этой папке находятся все шаблоны документов для автоматизации закупок по 44-ФЗ.

**Статус:** 
- ✅ Excel-шаблон НМЦК полностью работает
- ✅ Реальные Word-шаблоны добавлены (2026-06-08)
- ⏳ Требуется подготовка плейсхолдеров и интеграция с бэкендом

**См. также:**
- `TEMPLATES_MAPPING.md` - маппинг файлов и функций
- `WORD_TEMPLATES.md` - детальная информация по Word-шаблонам

## Генерация шаблона НМЦК (Excel)

### Способ 1: Python (рекомендуется)

```bash
# Установите зависимости
pip install openpyxl

# Запустите скрипт
python generate_nmcc_template.py
```

### Способ 2: Go

```bash
# Запустите скрипт
go run generate_template.go
```

### Способ 3: Вручную (Excel)

1. Создайте новый файл Excel
2. Настройте структуру согласно таблице ниже
3. Сохраните как `nmcc_template.xlsx`

## Структура шаблона

| Ячейка | Содержание | Тип | Пример/Формула |
|--------|------------|-----|----------------|
| A1 | Заголовок | Текст | "ОБОСНОВАНИЕ НАЧАЛЬНОЙ (МАКСИМАЛЬНОЙ) ЦЕНЫ КОНТРАКТА" |
| A4 | Наименование закупки | Плейсхолдер | `{{ProcurementTitle}}` |
| A5 | ИКЗ | Плейсхолдер | `{{IKZ}}` |
| A7:I7 | Заголовки таблицы | Текст | № п/п, Наименование, Ед. изм., ... |
| D8:F8 | Цены из КП | Число/Плейсхолдер | `{{Price1}}`, `{{Price2}}`, `{{Price3}}` |
| G8 | Средняя цена | Формула | `=AVERAGE(D8:F8)` |
| I8 | Итого по позиции | Формула | `=G8*H8` |
| I9 | ИТОГО НМЦК | Формула | `=SUM(I8)` |
| B19 | Коэф. вариации | Формула | `=(B18/AVERAGE(D8:F8))*100` |
| B20 | Однородность | Формула | `=IF(B19<=33, "ДА", "НЕТ")` |

## Плейсхолдеры для замены

Бэкенд Go будет заменять следующие переменные:

| Плейсхолдер | Описание | Источник |
|-------------|----------|----------|
| `{{ProcurementTitle}}` | Наименование закупки | Шаг 1 Wizard |
| `{{IKZ}}` | Идентификационный код закупки | Вводится вручную |
| `{{ItemName}}` | Наименование товара | Шаг 2 (ТЗ) |
| `{{UOM}}` | Единица измерения | Шаг 2 (ТЗ) |
| `{{Price1}}`, `{{Price2}}`, `{{Price3}}` | Цены из КП | Шаг 3 (НМЦК) |
| `{{Quantity}}` | Количество | Шаг 2 (ТЗ) |
| `{{Provider1_Name}}` | Название поставщика 1 | Шаг 3 (НМЦК) |
| `{{Provider1_INN}}` | ИНН поставщика 1 | Шаг 3 (НМЦК) |
| `{{Provider1_Date}}` | Дата КП 1 | Шаг 3 (НМЦК) |
| `{{ResponsibleName}}` | Ответственный | Шаг 1 (Комиссия) |
| `{{CurrentDate}}` | Текущая дата | Авто (Go) |

## Проверка шаблона

После генерации проверьте:

1. ✅ Формулы работают корректно
2. ✅ Ячейки с формулами не заблокированы
3. ✅ Формат ячеек числовой (для цен)
4. ✅ Формат даты для ячеек с датами
5. ✅ Печать настроена (A4, альбомная)

## Интеграция с бэкендом

Бэкенд использует библиотеку `excelize` для заполнения шаблона:

```go
f, err := excelize.OpenFile("templates/nmcc_template.xlsx")
if err != nil {
    return nil, err
}

// Замена плейсхолдеров
f.SetCellValue("Лист1", "B4", procurementTitle)
f.SetCellValue("Лист1", "D8", offer1.Price)
// ...

// Сохранение
buffer := new(bytes.Buffer)
f.Write(buffer)
```
